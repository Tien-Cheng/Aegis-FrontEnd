Introduction to Data Science

Description

Take your first step into the exciting Data Science field! This introductory course provides an overview of data science, a field which is in demand globally. Learn basic data science skills like data prepration, data modeling and data analysis. Participants can expect to learn the basic knowledge of data science and how data science can be applied in real life.

Course Details
Mode of Training Physical or Online
Duration 1 day (Physical, 10am - 7pm) / 2 days (Online, 2pm - 6pm)
Scheduling Physical:
  • August 2020 (15, 24)
  • October 2020 (5, 10, 14)
Online:
  • August 2020 (16, 20, 26)
  • September 2020 (1, 4)
  • October 2020 (2, 20, 25)

Prerequisites

  • Basic Mathematics
  • Basic Knowledge of Microsoft Excel

Content

  1. What is Data Science?
  2. Data Science Pipeline
  3. Data Aquisition
  4. Data Preparation
  5. Data Analysis and Visualization with Excel
  6. Linear Regression
  7. Applications of Data Science

Assessment

In addition to a test to be conducted at the end of the course, participants will attempt a simple data science project in groups to apply what they have learnt.
